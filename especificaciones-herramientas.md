# Especificaciones de Herramientas Interactivas - Blog de Finanzas Personales

## Objetivo

Crear herramientas interactivas que complementen el contenido del blog, aumenten el engagement y generen backlinks naturales.

---

## Herramientas Prioritarias (20 herramientas)

### Finanzas Personales (8 herramientas)

#### 1. Calculadora de Presupuesto Mensual
- **URL:** /herramientas/calculadora-presupuesto-mensual/
- **Inputs:** Ingresos, gastos fijos, gastos variables, ahorro objetivo
- **Outputs:** Balance, distribución por categoría, alertas de exceso
- **Features:** Plantilla descargable, comparación mes a mes, gráfico circular
- **Relacionadas:** Regla 50/30/20, Gastos hormiga, Ahorrar dinero

#### 2. Calculadora de Regla 50/30/20
- **URL:** /herramientas/calculadora-regla-50-30-20/
- **Inputs:** Ingreso mensual
- **Outputs:** 50% necesidades, 30% deseos, 20% ahorro
- **Features:** Comparación con gastos actuales, ajuste personalizable
- **Relacionadas:** Presupuesto mensual, Gastos hormiga

#### 3. Calculadora de Eliminación de Deudas
- **URL:** /herramientas/calculadora-eliminar-deudas/
- **Inputs:** Monto de deuda, tasa de interés, pago mensual
- **Outputs:** Tiempo para pagar, interés total, método bola de nieve vs avalancha
- **Features:** Comparación de métodos, plan de pago
- **Relacionadas:** Eliminar deudas, Interés compuesto

#### 4. Calculadora de Fondo de Emergencia
- **URL:** /herramientas/calculadora-fondo-emergencia/
- **Inputs:** Gastos mensuales, meses de cobertura deseados (3-6)
- **Outputs:** Monto objetivo, tiempo para alcanzarlo con ahorro mensual
- **Features:** Progreso, hitos de ahorro
- **Relacionadas:** Emergency Fund, Ahorrar dinero

#### 5. Calculadora de Gastos Hormiga
- **URL:** /herramientas/calculadora-gastos-hormiga/
- **Inputs:** Gastos diarios pequeños (café, snacks, suscripciones)
- **Outputs:** Total mensual, total anual, impacto en ahorro
- **Features:** Identificar gastos recurrentes, sugerencias de eliminación
- **Relacionadas:** Gastos hormiga, Presupuesto mensual

#### 6. Calculadora de Inflación
- **URL:** /herramientas/calculadora-inflacion/
- **Inputs:** Monto actual, tasa de inflación anual, años
- **Outputs:** Monto futuro ajustado, pérdida de poder adquisitivo
- **Features:** Comparación con diferentes tasas, gráfico
- **Relacionadas:** Economía global, Inversiones

#### 7. Calculadora de Interés Compuesto
- **URL:** /herramientas/calculadora-interes-compuesto/
- **Inputs:** Monto inicial, aporte mensual, tasa anual, años
- **Outputs:** Monto final, interés total, gráfico de crecimiento
- **Features:** Comparar con interés simple, tabla año por año
- **Relacionadas:** Inversiones, Eliminar deudas

#### 8. Calculadora de Poder Adquisitivo
- **URL:** /herramientas/calculadora-poder-adquisitivo/
- **Inputs:** Salario actual, tasa de inflación, años
- **Outputs:** Salario futuro ajustado, pérdida real
- **Features:** Comparar con canasta básica
- **Relacionadas:** Inflación, Salario, Economía global

## Especificaciones Técnicas Generales

### Stack Tecnológico
- **Frontend:** HTML, CSS, JavaScript (Vanilla)
- **Framework:** Integración con blog existente
- **Responsive:** Mobile-first design
- **Performance:** < 2s LCP, < 0.1 CLS

### Características Comunes
- Validación de inputs en tiempo real
- Cálculos instantáneos (sin botón "calcular")
- Exportar resultados a PDF (versión PRO)
- Guardar historial de cálculos (versión PRO)
- Compartir resultados en redes sociales
- FAQ Schema en cada página de herramienta
- Breadcrumb Schema
- Open Graph tags

### SEO
- URL amigables (/herramientas/nombre-herramienta/)
- Title optimizado: "Calculadora de X - [Nombre Blog]"
- Meta description descriptiva
- Canonical URL
- Datos estructurados SoftwareApplication

### Monetización
- Banner publicitario superior (no intrusivo)
- CTA a versión PRO para funciones avanzadas
- Enlaces a artículos relacionados con afiliados
- No más de 2 anuncios por página

### Analytics
- Event tracking para cada cálculo
- Medir tiempo en página
- Seguir herramientas más usadas
- Track CTAs a PRO

## Integración con el Blog

### Desde Artículos
- Cada artículo debe enlazar a 1-2 herramientas relevantes
- Ejemplo: Artículo "Cómo hacer un presupuesto" → Calculadora de Presupuesto Mensual

### Desde Herramientas
- Cada herramienta debe enlazar a 2-3 artículos relacionados
- Ejemplo: Calculadora de Presupuesto → Artículos sobre presupuesto, regla 50/30/20

### Lead Magnets
- Las herramientas pueden servir como lead magnets
- "Descarga tu plantilla de presupuesto" → Captura de email
